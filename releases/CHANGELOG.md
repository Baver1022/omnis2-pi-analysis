## [1.0.0] - 2026-01-07

### Added
- Initial release of OMNIS2 Pi Analysis
- 27 statistical tests implementation (NIST STS + TestU01 SmallCrush)
- Analysis orchestrator with modular architecture
- GPU acceleration support (optional CUDA)
- Comprehensive JSON output format
- Scientific reports (Polish and English PDFs)
- Mathematical formulas documentation
- Complete research results (55 JSON files)
